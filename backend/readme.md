# E-commerce Application

## Getting Started

### Installation procedure

npm i
npm i nodemon -g

### Environment Variable

##### DATABASE_USERNAME

##### DATABASE_PASSWORD

##### DATABASE_URL

##### JWT_SECRET

##### JWT_EXPIRES_IN

##### JWT_COOKIE_EXPIRES_IN

##### PORT=7890

##### NODE_ENV

##### STRIPE_SECRET_KEY

##### STRIPE_WEBHOOK_SECRET

##### Get started with the Stripe CLI

https://stripe.com/docs/stripe-cli

### Run the development server:

npm run dev
